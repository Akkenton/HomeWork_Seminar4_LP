## Задания и их пояснение в ДЗ от семинара №4 по ЛП

- **Задача №25**
    + Напишите цикл, который принимает на вход два числа (A и B) и возводит число A в натуральную степень B.  
    3, 5 -> 243 (3⁵)  
    2, 4 -> 16
- **Задача №27**
    + Напишите программу, которая принимает на вход число и выдаёт сумму цифр в числе.  
    452 -> 11  
    82 -> 10  
    9012 -> 12
- **Задача №29**
    + Напишите программу, которая задаёт массив из 8 элементов и выводит их на экран.  
    1, 2, 5, 7, 19 -> [1, 2, 5, 7, 19]  
    6, 1, 33 -> [6, 1, 33]  
    *В данном домашнем задании группа вместе с Мансуром Шафигуллиным приняли решение заполнять массив строго из 8 элементов рандомом в периоде от 0 до 100(не включительно: 0 - 99)*
